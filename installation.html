Overview

Running Ghost locally on your computer is straight forward, but requires that you install Node.js first.

Installing Node.js

Ghost requires node 0.10.*. To get it visit http://nodejs.org and press 'install' to download the latest version of Node.js.

Windows

The install button will download an .msi installer file. Double click this and follow the instructions to install Node.js as you would any other software.

Mac

The install button will download a .dmg. Install this as you would any other software.

Installing Ghost

Download the latest version of Ghost from Ghost.org. Unzip the archive to a memorable location.

  1. Open Terminal / Command Prompt
  2. Navigate to your Ghost directory
  3. Then enter npm install --production

Running Ghost

  1. In the Terminal / Command Prompt, type npm start

  2. This will have launched your Ghost blog, visit one http://localhost:2368/ to see
