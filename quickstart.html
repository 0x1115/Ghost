Overview

The Quickstart Guide to getting Ghost up and running is aimed at those of you who are already familiar with Node, or something similar like ruby on rails. If you're new in town, we recommend taking a look at the more in depth Installation Guide.

Get Ghost running locally

Ghost requires node 0.10.*. If you haven't already got it, head over to http://nodejs.org and download the latest version of Node.js. The installer will set up both Node and Node's excellent package manager, npm.

Download the latest version of Ghost from Ghost.org. Unpack the archive to a folder where you'd like to run Ghost - anywhere will do!

  1. Open Terminal / Command Prompt
  2. Navigate to your Ghost directory
  3. Then enter npm install --production
